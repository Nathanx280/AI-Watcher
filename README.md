# AI Offer Watcher

A desktop Python app that monitors RSS feeds and news sources for AI-related offers, free API credits, developer promotions, startup programs, and free AI tools. It automatically filters content using custom keywords, tracks previously seen posts, and sends alerts for new matching items. Includes a full Tkinter GUI, configurable feeds, desktop notifications, system tray support, logging, and background monitoring. Designed for continuously tracking AI-related deals and announcements from sources like Google AI, developer blogs, and news feeds.
